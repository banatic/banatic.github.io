# 데이터 통신과 네트워크 ( 5th edition)

{: .no_toc }


## **책의 구조**

<br>

1부 (1~2장) 에서는 **네트워크의 개요**를 설명한다

2부 (3~8장) 에서는 **물리층**에 대해 설명한다

3부 (9~17장) 에서는 **데이터링크**를 설명한다

4부 (18~22장) 에서는 **네트워크층**을 설명한다

5부 (23~24장) 에서는 **전송층**을 설명한다

6부 (26~29장) 에서는 **응용층**을 설명한다

7부 (30~32장) 에서는 **모든 계층과 관련된 주제** 그리고 **보안**에 대해 설명한다
