# 물리층 개요

정현파는 최대 진폭(peak amplitude), 주파수(frequency), 위상(phase)으로 구성된다

## 주기(period)

신호가 한 사이클을 완성하는데 필요한 시간의 양 (단위 : s(초))

## 주파수(frequency)

1초에 생성되는 신호 사이클의 수 (단위 : Hz)

한 정현파의 주파수는 그 파형이 가장 낮은 준위에서 가장 높은 준위로 움직이는 속도이다

주파수는 비록 초당 사이클의 수로 기술되긴 하지만 시간에 대한 신호의 변화율을 총량적으로 측정한 것이다

주파수와 주기의 관계

서로 역수 관계이다

$frequency = 1/period$

## 양극단

- 신호가 변화하지 않고 일정한 전압 준위가 전시간에 걸쳐 유지 된다면 이러한 경우 주파수는 0이다
- 반대로 신호가 한순간에 변화한다면 신호의 주기는 0이되고 주파수는 주기의 역이므로 주파수는 무한대가 된다

## 위상(phase)

- 시각 0시에 대한 파형의 상대적인 위치
- 시간축을 따라 앞뒤로 이동될 수 있는 파형이 있다면, 위상은 그 이동된 양을 뜻한다
- 첫 사이클의 상태를 표시한다
- 위상은 각도나 라디안으로 측정된다
- 360도의 위상 이동은 파형이 시간축을 따라 완전히 한 주기만큼 이동한 것에 해당된다

    EX) 180도의 위상이동은 반주기 90도의 위상이동은 1/4주기

## 파장(wave length)

단순 신호가 한 주기 동안 진행할 수 있는 거리

$파장 = (전파속도) * (주기) = (전파속도)/(주파수)$

$wave length = c/f$

## 복합 신호(composite signal)

여러개의 단순 정현파들로 구성된다

> 푸리에 해석(Fourier analysis)

⇒ 복합 신호는 실제로는 서로 다른 주파수, 진폭, 위상을 가지는 다수의 정현파가 합쳐져서 만들어진 것이다

복합신호는 주기적일 수도 있고 비주기적일 수 도 있다

주기 복합 신호는 주파수가 서로 정수배 값을 갖는 순차적인 단순 정현파들로 분해될 수 있다

비주기 복합 신호는 연속적인 실수 값을 갖는 주파수와 진폭의 무한한 정현파들로 구성된다

주기 복합 신호와 같은 주파수 값을 가지는 주파수 값을 기본 주파수 혹은  첫째 조파(harmonic)이라고 부른다

n배수에 따라 제 n조파라고 부른다

## 대역폭(bandwidth)

여기서 말하는 대역폭은 헤르츠단위의 대역폭이다

복합 신호의 대역폭은 신호에 포함된 최고 주파수와 최저 주파수의 차이이다

## 비트율(bit per seconds)

1초동안 전송된 비트의 개수

## 비트 길이(bit length)

한 비트가 전송매체를 통해 차지하는 길이

bit length = propagtion speed * bit duration

$bitlength = propagationspeed*bitduration$

## 디지털 신호 전송

디지털 신호는 무한대 대역폭을 갖는 복합 아날로그 신호이다

### 기저대역 전송(baseband transmission)

디지털 신호를 아날로그 신호로 바꾸지 않고 있는 그대로 채널을 통해 전송하는 것이다

주파수 0부터 시작하는 대역폭을 갖는 저대역-통과 채널(low-pass channel)이 필요하다

기저대역 전송 방법에는 넓은 대역폭 전송과 좁은 대역폭 전송이 있다

N비트율을 가지고 있다고 할때, 

요구 대역폭 = N/2-0 = N/2 이다

### 광대역 전송(broadband transmission) - 변조를 이용한다

광대역 전송이란 디지털 신호로 전송하기 위해 아날로그 신호로 전환하여 전송하는 것을 말한다

변조를 하면 띠대역 통과 채널(bandpass channel)을 사용하여 전송할 수 있는데 띠대역 통과 채널은

주파수 0부터 시작하지 않는다

## 전송장애

1. 감쇠(attentuation)
2. 일그러짐(distortion)
3. 잡음(noise)

### 감쇠(attentuation)

에너지 손실

신호가 매체를 통해 이동할 때, 매체의 저항을 이겨내기 위해서 약간의 에너지가 손실된다.

증폭기를 사용하여 해결한다

데시벨(dB)

2개의 다른 점에서 두 신호 또는 하나의 상대적인 신호 세기를 측정한다

dB = 10log(P2/P1) 

P1,P2는 한 점의 전력이다

전력 = 전류*전압

전류 = 전압/저항 (옴의 법칙)

전력 = (전류)^2/저항

### 일그러짐(distortion)

신호의 모양이나 형태가 변하는 것을 의미한다

### 잡음(noise)

신호-대-잡음비(signal to noise ratio) SNR

→잡음 전력에 대한 신호 전력의 비

SNR = (average signal power)/(average noise power)

SNR도 전력의 비를 나타낸 것이기 때문에 dB로 표현 가능

SNRdB = 10log(SNR)

## 데이터 전송률의 한계

데이터 전송률에 영향을 미치는 것들

1. 가용 대역폭
2. 사용 가능한 신호 준위 개수
3. 채널의 품질 (잡음 정도)

## 데이터 전송률 계산 수식

잡음이 없을때→ 나이퀴스트 수식

잡음이 있을때→ 섀넌 수식

### 무잡음 채널 : 나이퀴스트 비트율

→잡음이 없는 채널의 최대 전송률 정의

bitrate = 2*bandwidth*log2(L)

bitrate = 초당 비트수(bps)

bandwidth = 채널의 대역폭

L = 신호 준위의 개수

신호 준위를 늘리면 시스템의 신뢰도를 떨어뜨릴 수 있다

### 잡음 채널 : 섀넌 용량

capacity = bandwidth * log2(1+SNR)

capacity = bps 단위의 채널 용량

bandwidth = 채널의 대역폭

SNR = 잡음에 대한 신호비

섀넌 수식에는 신호의 준위의 개수가 포함되지 않는다

이는 몇개의 준위를 사용하든 채널의 전송한계 이상의 전송률을 달성할 수 없다는 것을 의미한다

섀넌 용량은 상한값을 알려주고 나이퀴스트 수식은 몇개의 신호 준위가 필요한지를 알려준다

## 성능 (네트워크의 성능)

### 대역폭(bandwidth)

헤르츠 단위의 대역폭

→ 복합 신호에 포함된 주파수 영역 또는 채널이 통과 시킬 수 있는 주파수 영역

비트율 단위의 대역폭

→링크 또는 네트워크가 통과 시킬 수 있는 초당 비트 수

헤르츠 단위 대역폭과 비트율 단위 대역폭의 관계

→기본적으로 헤르츠 단위의 대역폭이 늘어나면 비트율 단위의 대역폭도 늘어난다

(뒤에서 조금더 자세히 알 수 있을 듯 하다)

### 처리율(throughput)

→ 어떤 지점을 데이터가 얼마나 빠르게 지나가는 지를 측정

비트율 단위의 대역폭과 구분된다

→ 어느 링크가 Bbps의 대역폭을 가질 수 있지만

이 링크를 사용하여 항상 B보다 작은 Tbps의 처리량만 가질 수 있다

대역폭은 잠재 성능치를 말해주고 처리율은 실제 전송 속도를 말해준다

### 지연(delay)

→ 첫번째 비트가 목적지를 향해 떠난 후에 온전히 전체 메세지가 모두 목적지에 도착할 때 까지 소요된 시간

지연 = 전파시간 + 전송시간 + 큐시간 + 처리시간

전파시간(propagation time)

→비트가 발신지로 부터 목적지까지 이동하는데 소요되는 시간

전파시간 = 거리 / 전파속도

### 전송시간(transmission time)

→ 첫 비트가 떠나고 마지막 비트까지 도착할때 까지의 시간

전송시간 = 메세지크기 / 대역폭

### 큐시간(queing time)

→도중 또는 종단 장치들이 메세지를 처리하기 전까지 메세지를 가지고 있는 시간

## 대역폭 - 지연 곱

→링크를 채울 수 있는 최대의 비트 개수
