# chapter4 관계대수와 SQL

# 관계대수

## 관계 연산자

## 셀렉션 연산자 (selection) 시그마 기호로 표시

- 한 릴레이션에서 selection condition을 만족하는 투플들의 집합을 생성한다.
- 단항 연산자이다
- 투풀을 골라내기 때문에 중복이 발생할 수 있다.
- 결과 릴레이션의 차수는 입력 릴레이션의 차수와 같다.
- 결과 릴레이션의 카디날리티는 항상 원래 릴레이션의 카디날리티보다 작거나 같다.
- 셀렉션조건 (predicate)는 부울식으로 구성된다.

## 프로젝션 연산자 (projection) 파이 기호로 표시

- 한 릴레이션의 애트리뷰트들이 부분 집합을 구한다.
- 이 애트리뷰트들은 애트리뷰트 리스트에 열거된 순서와 동일하게 결과 릴레이션에 나타난다.
- 애트리뷰트의 부분집합이기 때문에 튜플의 중복이 발생할 수 있다. 이를 제거해야 한다.

## 집합 연산자

- 집합 연산자의 입력으로 사용되는 두 개의 릴레이션은 합집합 호환(union compatible) 이어야 한다.
- 두 릴레이션 R1(A1, A2, ... An) R2(B1, B2, ... Bm)이 합집합 호환인 필요충분조건은 n==m 이고, 1 ≤ i ≤ n에 대해 domain(Ai) == domain(Bi) 이다.
- 두 릴레이션의 애트리뷰트가 같고 대응되는 애트리뷰트들의 도메인이 같다는 것이다.
- 기능은 집합 연산과 동일하다 ( 합집합 연산, 차집합 연산, 교집합 연산이 있다.)

 

## 카티션곱 연산자

- 카디날리티가 i인 릴레이션 R(A1, A2, ... An)과 카디날리티가 j인 릴레이션 S(B1, B2, ... Bm)의 카티션 곱 R X S 의 차수는 n + m 이고 카디날리티가 i * j 이고 애트리뷰트가 (A1, A2 ... An, B1, B2, ... Bm) 이며 R과 S의 튜플들의 모든 가능한 조합으로 이루어진 릴레이션이다.
- 튜플들의 결합을 말하는 것이다.

## 관계대수의 완전성

- 임의의 질의어가 필수적인 관계대수 연산자들 만큼의 표현력을 갖고 있으면 관계적으로 완전(relationaly complete) 하다고 말한다.

아래로는 필수적인 연산자는 아니다.

 

## 조인 연산자

- 두개의 릴레이션으로 부터 연관된 튜플들을 결합하는 연산자이다.
- 결과만 보면 카티션 곱을 수행한 후, 실렉션을 적용한 것과 같다. 하지만 비효율 적이기 때문에 다른 알고리즘 으로 조인한다.

### 세타조인과 동등조인

- 세타 조인을 수행하기 위해서 양쪽 릴레이션의 애트리뷰트들이 세타조인 조건을 만족하는 투플들만 골라낸다. 그리고 카티션곱과 비슷한 연산을 한다. 이것이 카티션 곱으로 조인을 하는 방법과의 차이점이다.
- 동등 조인은 세타 조인 중에서 비교 연산자가 = 인 조인이다.

### 자연조인

- 동등 조인의 결과 릴레이션에서 조인 애트리뷰트 한 개를 제외한 것을 자연 조인이라고 한다.
- 기호는 * 기호를 쓴다.

## 디비전 연산자

- 차수가  n + m 인  릴레이션 R(A1, A2, ... An, B1, B2, ... Bm)과 차수가 m인 릴레이션 S(B1, B2, ... Bm)의 디비전 연산의 결과는 차수가 N이고 S에 속하는 모든 투플 u에 대하여 투플 tu가 R에 존재하는 투플 t의 집합이다.

## 관계대수의 한계

1. 관계대수는 산술연산을 할 수 없다.
2. 집단 함수 (AVG, SUM ... )을 지원하지 않는다.
3. 정렬을 나타낼 수 없다.
4. 데이터베이스를 수정할 수 없다.
5. 프로젝트 연산의 결과에 중복된 투플을 나타내지 못한다.

## 새로운 관계대수 연산자

1. 집단 함수 (AVG, SUM, ... etc)
2. 그룹화
    - 릴레이션은 그룹화 애트리뷰트들의 값에 의해서 분할되고, 각 그룹에 집단함수를 적용할 수 있다.
3. 외부조인(outer join)
    - 상대 릴레이션에서 대응되는 투플을 갖지 못하는 투플이나, 조인 애트리뷰트에 NULL값이 들어있는 투플을 다루기 위해서 원래의 조인 연산을 확장한 연산이다.
    - 외부조인은 두 릴레이션에서 대응되는 투플들을 결합하면서, 대응되는 투플을 갖지 않는 투플과 조인 애트리뷰트에 NULL갑승ㄹ 갖는 투플도 결과 릴레이션에 포함 시킨다.
    - 왼쪽, 오른쪽, 완전 외부조인이 있다.
    - 왼쪽은 무조건 표현, 오른쪽은 무조건 표현, 양쪽 무조건 표현한다는 의미이다.
    - 애트리뷰트 값이 널 값을 가지는 것을 각오하고 조인하여 표현하는 연산자이다.
