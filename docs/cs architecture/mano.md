# Mano의 컴퓨터시스템구조(제 3판)
{: .no_toc }

## **책의 구조**


1장부터 4장까지는 컴퓨터 조직과 설계에 사용되는 **디지털 구성품**을 소개한다.

5장부터 7장까지는 **컴퓨터를 설계하고 프로그램하는데 필요한 기술**을 설명한다.

8장에서 9장까지는 **중앙 처리 장치(CPU)의 구조**를 설명한다.

11장부터 12장까지는 **입출력 프로세서(IOP)와 메모리 장치(RAM)**을 설명한다.
