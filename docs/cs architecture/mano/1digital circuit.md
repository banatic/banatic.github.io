# 논리 게이트
**게이트**란 논리 회로로서 입력 논리에 따라 0, 1의 신호를 만드는 하드웨어다. 

AND, OR, Inverter, Buffer, NAND, NOR, XOR, XNOR 등등의 회로가 존재한다.

# 부울 대수
**부울 대수**란 **이진 변수와 논리 동작**을 취급하는 대수이다.
